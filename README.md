#  onwakeup
trigger callback when computer wakes up from sleep/hibernation

## Easy to use

#### Module usage
```javascript
const onwakeup = require( 'onwakeup' )
onwakeup( function ( delta ) {
  console.log( 'Wakeup! Computer was asleep for ' + delta + ' milliseconds' )
} )
```
